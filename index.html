Tentang Aku

Halo, aku Fadhil

Aku adalah pelajar SMKN 2 Mojokerto yang punya ketertarikan besar di dunia pengembangan web, sistem operasi Linux, dan integrasi perangkat keras (hardware). Sejak mulai belajar dunia IT, aku selalu tertantang untuk menggabungkan kode perangkat lunak dengan infrastruktur fisik agar sistem bisa berjalan mandiri.

Aku percaya bahwa teknologi tidak harus mahal. Melalui eksperimen dengan komputer mini / STB bekas, aku berhasil merakit **Mini Private Server** mandiri di rumah untuk kebutuhan penyimpanan data lokal gratis, manajemen iklan jaringan, hingga hosting server portofolio lokal.

Selain mengulik server dan hardware, aku juga aktif mendesain UI di Figma serta mengimplementasikannya ke dalam kode web responsif.
